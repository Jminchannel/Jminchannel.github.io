File类与IO流

概述

  • File类和相关流都定义在java.io包下
  • File对象代表硬盘或网络中可能存在的一个文件或文件目录
  • File能新建、删除、重命名文件和目录，但File不能访问文件内容本身（需要使用输入输出流）
    • File对象可以作为参数传递给流的构造器，成为流对象

列出目录的下一级

  • public String[] list() ：返回一个 String 数组，表示该 File 目录中的所有子文件或目录。
  • public File[] listFiles() ：返回一个 File 数组，表示该 File 目录中的所有的子文件或目录。

File 类的重命名功能
